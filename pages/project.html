Course Manager - jQuery

This is a course management system that implements a Java middle tier using Spring Boot and jQuery. This is the first of several projects that together will build on each other where faculty can author courses and students can register for online tutorials.

In this project the admin users are able to manage other users of a multi user Web Application. System administrators are able to list, create, update, and remobe users.

Course Manager - React

The Course Manager application is divided into three parts, the admin application, the faculty application, and the student application. This application focuses on the faculty application. This is a React client that uses the same middle tier established in the first project, and just focuses on adding support for the new React client.
